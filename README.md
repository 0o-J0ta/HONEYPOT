# HONEYPOT
🍯 Um laboratório prático de Defesa Cibernética focado na implantação do Honeypot Cowrie em ambiente isolado. Desenvolvido para capturar tentativas de invasão via SSH/Telnet em tempo real, coletar inteligência de ameaças (Threat Intel) e realizar análise forense de logs e comandos executados por atacantes.
